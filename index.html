Analisis Ulasan Eleganza Dress

Model Machine Learning

Menggunakan model Logistic Regression untuk mengklasifikasikan ulasan menjadi: positif, netral, dan negatif.

Parameter Utama Model

  • C = 1.0
  • Max Iter = 200
  • Vectorizer = TF-IDF (max_features=2000)

Ulasan Pelanggan

Ulasan Baju Wisuda

"Warnanya cantik dan bahan adem dipakai." → Positif

"Lumayan, tapi pengiriman agak lama." → Netral

"Ukuran tidak sesuai dan bahan terasa panas." → Negatif

"Detail bordir elegan dan terlihat mewah." → Positif

"Resleting mudah rusak, tidak sesuai harga." → Negatif

"Modelnya standar saja." → Netral
